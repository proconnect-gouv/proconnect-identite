# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in **ProConnect Identité**, please **do not open a public GitHub issue**.

Report it by email to:

* **[soc.rie@pm.gouv.fr](mailto:soc.rie@pm.gouv.fr)**
* **[dinum.ssi@pm.gouv.fr](mailto:dinum.ssi@pm.gouv.fr)**

You may write in French or English.

Please include:

* A clear description of the issue
* Steps to reproduce (if applicable)
* Potential impact
* Affected versions
* Your contact information

---

## Responsible Disclosure

We ask researchers to:

* Avoid public disclosure before a fix is deployed
* Limit testing to what is strictly necessary
* Not access, modify, or delete data

We commit to:

* Acknowledging your report
* Investigating promptly
* Keeping you informed when possible

---

## Scope

This policy applies to the source code and related components of this repository.
